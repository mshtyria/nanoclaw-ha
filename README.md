# NanoClaw Home Assistant Add-ons

Home Assistant add-on repository for [NanoClaw](https://nanoclaw.dev) — a secure, lightweight personal AI agent with container isolation.

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**
2. Click the three-dot menu (⋮) in the top right → **Repositories**
3. Add this repository URL:
   ```
   https://github.com/qwibitai/ha-nanoclaw
   ```
4. Find **NanoClaw** in the store and click **Install**

## Add-ons

### NanoClaw

Personal AI agent that connects to WhatsApp or Telegram and runs Claude-powered tasks in isolated containers.

- Messenger connectivity (WhatsApp, Telegram)
- Container isolation per session
- Scheduled tasks (cron)
- Agent swarms for complex tasks
